다만, 데이터 명세서 상에 LiDAR 센서의 원점(Origin) 좌표계 정보가 실제 데이터와 약간의 오차가 발생하는 것 같아 확인 요청드립니다.

2. 차량 무게 중심점 대비 센서 위치 좌표값을 다시 한번 확인 부탁드립니다.

안녕하세요. 구미시 자율주행 관제센터입니다.

먼저 저희 데이터를 활용해주셔서 감사드립니다. 문의하신 내용에 대해 다음과 같이 답변드립니다.

1. 제공되는 LiDAR 원천 데이터의 좌표계는 제안하신 ISO 8855 규격을 기반으로 작성되었습니다.
2. 지난 3월 센서 캘리브레이션 업데이트 과정에서 발생한 미세한 오차가 확인되어, 수정된 센서 설치 좌표(Offset) 데이터를 별도로 보내드렸습니다.

추가적인 문의사항이 있으시면 언제든지 연락 부탁드립니다.
