자율주행 데이터 제공

혼합 주행 분석 데이터

자율차와 일반 차량이 공존하는 복잡한 도로 환경에서의 교통 흐름 안정성 분석 리포트를 제공합니다.

🖥️

실시간 관제 데이터

디지털 트윈 모델을 기반으로 센터에서 상시 모니터링 중인 구미 구역의 실시간 교통 현황 및 이벤트 데이터를 전송합니다.

🏢

기업 지원 서비스

자율주행 기술 실증이 필요한 기업을 위해 데이터 가공 및 기술 멘토링 서비스를 지원합니다.
